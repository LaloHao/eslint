** Usage
   Root folder should look like this:

   #+BEGIN_SRC sh
     .
     ├── node_modules
     ├── package.json
     └── .eslintrc.yaml
   #+END_SRC
*** Install

    #+BEGIN_SRC sh
      yarn add --dev lalohao/eslint#develop
    #+END_SRC

*** Extend
    Add =.eslintrc.yaml= to project root.

    #+BEGIN_SRC yaml
      extends: hao
    #+END_SRC
